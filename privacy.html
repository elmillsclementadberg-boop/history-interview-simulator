隐私说明

生效日期：2026-02-24

1. 收集与处理范围

你在页面输入的文本、语音转写请求会发送至本站后端，再由后端调用第三方模型服务（Groq API）进行处理。

2. 我们本地存储的数据

浏览器会在 localStorage 中保存页面配置（如接口地址、模型、你自行填写的 API Key）。这些数据保存在你的浏览器本地，不会自动上传到本站数据库。

3. 服务端数据

服务端默认不存储完整问答内容，仅维护限流计数（内存或 Redis）。限流数据用于防刷和预算控制。

4. 第三方处理

请求会由 Groq API 处理。第三方服务的数据处理规则以其官方文档和法律条款为准。

5. 用户注意事项

请勿提交敏感个人信息、违法内容或侵权内容。你提交内容即表示你有权使用并同意本说明。
